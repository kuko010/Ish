## Publishing to GitHub

To publish your project, set up the following:

1. Create a `.env` file at the project root:
```env
GITHUB_TOKEN=ghp_yourtoken
```
2. Update the `github_repo` property in [`gradle.properties`](gradle.properties) to point to your repository path.
